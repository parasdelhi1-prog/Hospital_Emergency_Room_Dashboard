# Hospital_Emergency_Room_Dashboard
Advanced Excel project analyzing 9,216 patient records with interactive dashboard for healthcare decision-making

**An Advanced Excel Analytics Project**

As part of my analytics learning journey, I developed an Advanced Excel project focused on building a Hospital Emergency Room Dashboard. The aim was to analyze patient records and operational metrics, then present them through an interactive Excel dashboard that supports healthcare decision making. This project allowed me to apply advanced Excel techniques such as pivot tables, charts, conditional formatting, and dashboard design to a realistic healthcare dataset.

---

## 🔍 Dashboard Highlights

**The dashboard provides a comprehensive monthly report of emergency room performance:**

| Metric | Value |
|--------|-------|
| Total Patients Analyzed | 9,216 |
| Average Wait Time | 35.26 minutes |
| Patient Satisfaction Score | 4.99/10 |
